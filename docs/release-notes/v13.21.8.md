# v13.21.8 - Self-Target Abilities & Display Fixes

**Release Date**: February 2026

## Patch Releases (v13.21.6 - v13.21.8)

This series of patch releases focused on resolving issues with self-target action cards, fixing damage type display problems, and improving localization following the v13.21.0 feature release.

## Key Highlights

- **Self-Target Action Card Fix**: Resolved one of the big potential issues with self-target abilities by ensuring target problem checks are skipped for self-target abilities
- **Damage Type Display Fix**: Corrected issue where saved damage types weren't displaying correctly when items still contained embedded item data
- **Localization Improvements**: Moved untranslated strings to proper sources and added source strings for better translation support

## Bug Fixes and Improvements

### Critical Fixes

#### Self-Target Action Card Validation (v13.21.8)

- **Issue**: Self-target abilities were incorrectly triggering target problem checks, causing potential issues during execution
- **Fix**: By ensuring that we do not check for target problems on self-target abilities, we're resolving one of the big potential issues with self-target abilities
- **Impact**: Self-buff, self-healing, and stance-change action cards now execute without unnecessary validation errors
- **Pull Request**: #152

#### Damage Type Display with Embedded Data (v13.21.8)

- **Issue**: Saved damage types were not displaying correctly if items still contained embedded item data
- **Fix**: Corrected the display logic to properly handle damage type rendering when embedded item data is present
- **Impact**: Damage type information now displays correctly in all scenarios, improving visibility and user understanding

### Localization

#### Translation String Updates (v13.21.8)

- **Improvement**: Moved untranslated strings to proper source files
- **Additional**: Added source strings to enable proper translation workflow
- **Impact**: Improves translation support and ensures user-facing text can be properly localized
- **Pull Request**: #151

## Version Information

- **Release Version**: 13.21.8
- **Foundry Compatibility**: v13+ minimum, v13+ verified
- **Focus**: Self-target abilities, display fixes, and localization
- **Previous Version**: v13.21.5 introduced code refactoring & bug fixes
